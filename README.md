# WeatherApp
A responsive weather web application built with HTML, CSS, and JavaScript. Features real-time weather data from a weather API with location-based forecasts, temperature units conversion, and intuitive UI design.

A fully-functional weather application with real-time data fetching! Features include:
✅ Live weather updates from API
✅ Search any city worldwide
✅ 5-day forecast display
✅ Temperature unit conversion
✅ Responsive mobile-first design
✅ Clean, modern UI with CSS animations
